Acerca de esta aplicación

Esta es una aplicación creada para recibir de manera segura donaciones a través de Mercado Pago. Fue desarrollada con el fin de aprender a usar las pasarelas de pago en plataformas e-commerce, aunque también se podría usar este desarrollo en algún tipo de Sociedad o Movimiento sin fines de lucro que realice un bien a personas que necesiten ayuda.
